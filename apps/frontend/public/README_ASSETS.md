# Statische Assets

Platzieren Sie hier Ihre statischen Dateien:

## Favicon
- **favicon.ico** - Hauptfavicon (16x16, 32x32, 48x48)
- **favicon.png** - PNG-Version des Favicons
- **apple-touch-icon.png** - Für iOS (180x180)
- **icon.png** - Für moderne Browser (512x512)

## Logo
- **logo.png** - Hauptlogo (empfohlen: transparent, mindestens 200px Breite)
- **logo-dark.png** - Logo für Dark Mode (optional)
- **logo-small.png** - Kleine Version für Header (optional)

## Weitere Assets
- **og-image.jpg** - Open Graph Bild für Social Media (1200x630)
- **twitter-image.jpg** - Twitter Card Bild (1200x630)

Nach dem Hinzufügen der Dateien werden sie automatisch unter `/logo.png`, `/favicon.ico` etc. verfügbar sein.
